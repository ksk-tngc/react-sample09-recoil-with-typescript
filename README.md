## 概要

React + TypeScript で Recoil を用いた状態管理のサンプルです。

以下の要素を含みます：

* TypeScript
* Chakra UI
* Atomic Design
* Recoil (Global State)
* カスタムフック
* 再レンダリングの最適化 (memo, useCallback)
* Fontsource
* React Icons

※ [Context を使用した状態管理](https://github.com/ksk-tngc/react-sample08-todo-app-using-typescript-and-chakra-ui)を Recoil に変更したサンプルになります。
